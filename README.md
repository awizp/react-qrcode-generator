# QR Code Generator | React + TailwindCSS

A clean QR Code Generator built using React and TailwindCSS.

### Languages used

* React: The structure of the web page.

* TailwindCSS: To fetch modern styles to clean look.

### Installation

1. Clone this repository to your local machine

``````git clone https://github.com/awizp/react-qrcode-generator.git``````

2. Open the folder and Run ```npm install``` for install debendencies.

3. Run ```npm run dev``` to run the application in your local host.
